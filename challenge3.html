Welcome to the cryptography challenge! Your task is to decode a message that has been encrypted using the ROT13 cipher.

Encrypted Message:

Gur synt vf: VRRR{ebg13_znfgeryl_haybpxrq}

Need a hint?

ROT13 is a simple letter substitution cipher that replaces a letter with the 13th letter after it in the alphabet.

For example: A → N, B → O, C → P, ..., M → Z, N → A, O → B, ..., Z → M

You can use CyberChef with ROT13 to decode the message.

Or in Python: import codecs; codecs.encode('your_text', 'rot13')
