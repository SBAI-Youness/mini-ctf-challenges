ROT13 Cipher Challenge

Welcome to the cryptography challenge! Your task is to decode a message that has been encrypted using the ROT13 cipher, a simple letter substitution cipher that replaces each letter with the 13th letter after it in the alphabet.

Encrypted Message

Gur synt vf: VRRR{ebg13_znfgeryl_haybpxrq}

Need a hint?

ROT13 is a special case of the Caesar cipher. Each letter is replaced by the letter 13 positions after it in the alphabet. For example:

  • A → N
  • B → O
  • M → Z
  • N → A
  • Z → M

You can use online tools like CyberChef with the ROT13 function to decode the message.

Or use this Python one-liner: import codecs; print(codecs.encode('your_text', 'rot13'))
